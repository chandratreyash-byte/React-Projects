# 📚 Book Record Management System (Mini CRUD App)

A simple **Book Record Management System** built using the **MERN Stack** that allows users to perform basic **CRUD (Create, Read, Update, Delete)** operations on book records.

---

## 🚀 Features

- Add new book records  
- View all available books  
- Update existing book details  
- Delete book records  
- Responsive user interface  
- RESTful API architecture
- Role based Authentication (JWT)

---
## Backend Setup

- cd server
- npm install
- npm start

---
## Frontend setup

- cd client
- npm install
- npm run dev

---
## 🛠️ Tech Stack

### Frontend
- React.js  
- Tailwind CSS  
- Axios  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB
